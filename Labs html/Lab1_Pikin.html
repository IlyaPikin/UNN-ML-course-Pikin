Ошибка 1.¶

Если гиперпараметр k (число кластеров) заранее неизвестен, то алгоритм выдаёт неверные результаты кластеризации.

In [59]:
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans

df1 = pd.read_csv(r'C:\Users\Илья\Desktop\Лабы АРО\Лаба 1\basic5.csv')
data1 = np.array(df1.iloc[:,0:2], dtype=float)
true_colors1 = np.array(df1.iloc[:,2], dtype=int)


kmeans = KMeans(n_clusters=4, init='k-means++', n_init=10)
kmeans.fit(data1)

fig, axes = plt.subplots(1, 2)

axes[0].set_title('Размеченные множества')
axes[0].scatter(data1[:,0], data1[:,1], c=true_colors1, cmap='rainbow', marker='.')

axes[1].set_title('Результат кластеризации k-means')
axes[1].scatter(data1[:,0], data1[:,1], c=kmeans.labels_, cmap='rainbow', marker='.')

axes[1].scatter(kmeans.cluster_centers_[:,0] , kmeans.cluster_centers_[:,1],
                color='black', marker='*')

Ошибка 3.¶

Результат кластеризации зависит от выбора исходных центров кластеров, а их оптимальный выбор, вообще говоря, неизвестен. В примере ниже выбраны центры со случайными координатами (неудачный выбор), которые обозначены черными звёздочками.
